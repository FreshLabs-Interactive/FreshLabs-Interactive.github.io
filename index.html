RepoBridge
Built by developers, for developers.

RepoBridge puts full Git version control directly inside the Unity Editor. Stage files, write commits, manage branches, lock binary assets, and sync with your remote server — all without leaving Unity. Modelled closely on GitHub Desktop for a familiar, approachable interface.

  • Full Git staging, committing & push/pull from a native Editor window
  • Branch creation, switching, and merge management
  • Binary asset locking to prevent team conflicts on .fbx, .png, and more
  • Familiar GitHub Desktop-inspired interface — minimal learning curve
  • Works with any remote Git server: GitHub, GitLab, Bitbucket, self-hosted

Toast XML System
Built by developers, for developers.

Bring native-style Windows toast notifications into your Unity game with a fully interactive Editor UI. Define notifications in XML, configure timing, styling, and triggers — then preview everything directly inside the Unity Editor without ever leaving your project.

  • Windows-style toast notifications for in-game use
  • XML-driven notification definitions — no code required for basic setups
  • Interactive Editor UI for building and previewing notifications live
  • Configurable display duration, position, icon, title, and body text
  • Event callbacks for user interaction (dismiss, click, timeout)
